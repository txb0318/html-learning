5. 删除临时目录及文件  
>不能直接使用delete命令删除存在文件的目录，需先将目录下的文件删除后，再删除该目录
```J
    File dir = new File(path);
    File[] listFiles = dir.listFiles();
    if(listFiles != null)
    {
        for(File file : listFiles)
        {
            file.delete();
        }
    }
    dir.delete();
```
